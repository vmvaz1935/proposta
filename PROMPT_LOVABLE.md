Criar uma landing page profissional para vender pacotes jurídicos para contadores. O escritório de advocacia oferece serviços de blindagem jurídica escalável para escritórios contábeis.

## Objetivo da Landing Page

Criar uma landing page de conversão para vender pacotes jurídicos para contadores, com:

- Percepção de valor (benefícios + dores que resolve)
- Estrutura comercial pronta para reunião / envio em link

## Diretrizes de Design

### Estilo Visual
- Design moderno, limpo e profissional
- Visual "premium" e corporativo
- Tipografia profissional com boa legibilidade
- Espaçamento generoso
- Cores: Azul escuro institucional (#0f172a, #1e293b), azul accent (#3b82f6), dourado (#f59e0b)
- Fundo claro com seções alternadas (branco/cinza muito claro)

### Elementos Visuais
- Cards com bordas arredondadas (12-16px)
- Sombras suaves e profundidade
- Badges coloridos para níveis (Simples/Médio/Robusto)
- Ícones discretos (SVG inline ou emojis) apenas para reforçar leitura
- Animações sutis no hover
- Transições suaves

### Responsividade
- Design totalmente responsivo (desktop, tablet, mobile)
- Grid que se adapta: 4 colunas → 2 colunas → 1 coluna
- Navegação mobile-friendly
- Textos legíveis em todas as telas

### Navegação
- Header fixo com logo e menu
- Menu de navegação com links para seções:
  - Para Quem É
  - Documentos
  - Pacotes
  - Planos
  - B2B2C
  - Contato
- Scroll suave entre seções

## Qualidade de Copy

### Tom
- Empresarial, direto, "estratégico"
- Sem promessas de resultado
- Linguagem simples para o contador entender

### Destaques
- Reduzir retrabalho
- Reduzir risco
- Padronizar atendimento
- Escalar sem perder controle
