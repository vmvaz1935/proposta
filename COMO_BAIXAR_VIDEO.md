# Como Baixar o Vídeo de Ponte Estaiada

## Opção 1: Download Manual do Pexels

1. Acesse: https://www.pexels.com/video/suspension-bridge-2491284/
2. Clique no botão "Download" (ícone de download)
3. Escolha a resolução HD (1920x1080) ou UHD (2560x1440)
4. Salve o arquivo como `ponte_estaiada.mp4` na mesma pasta do `index.html`

## Opção 2: Usar Pixabay

1. Acesse: https://pixabay.com/videos/search/suspension%20bridge/
2. Escolha um vídeo de ponte estaiada
3. Baixe em resolução HD
4. Renomeie para `ponte_estaiada.mp4` e coloque na pasta do projeto

## Opção 3: Usar Coverr

1. Acesse: https://coverr.co/
2. Busque por "bridge" ou "suspension bridge"
3. Baixe o vídeo
4. Renomeie para `ponte_estaiada.mp4` e coloque na pasta do projeto

## Formato Recomendado

- **Resolução**: 1920x1080 (HD) ou superior
- **Formato**: MP4
- **Codec**: H.264
- **Nome do arquivo**: `ponte_estaiada.mp4`
- **Localização**: Mesma pasta do `index.html`

## Após Baixar

1. Coloque o arquivo `ponte_estaiada.mp4` na pasta do projeto
2. O HTML já está configurado para usar este arquivo local
3. Se o arquivo local não existir, usará as URLs do Pexels como fallback

## Otimização (Opcional)

Se o arquivo estiver muito grande (>10MB), você pode comprimir usando:
- HandBrake (gratuito)
- FFmpeg
- Online: https://www.freeconvert.com/video-compressor

Mantenha a qualidade em pelo menos 80% para não perder muito detalhe.
